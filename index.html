Introduction

This website contains data and code of the benchmark evaluation of online visual tracking algorithms.
 You can find the following resources from this site.
  • Benchmark results
  • Dataset with ground-truth annotations
  • Large-scale: 1,550 sequences
  • High-quality: Manual annotation with careful inspection in each frame
Please check out the benchmark details and download links at WordAnimal Benchmark page, evaluation toolkit and sample results at Evaluation and Result

How to get the tracker benchmark data

Because our data sets is too large, it is divided into three parts in Google Drive:(WATB_part1.zip、WATB_part2.zip、WATB_part3.zip、)
 Challenge attribute set in Google Drive:(WATB_attribute_files.rar)
