Introduction

This website contains data and code of the benchmark evaluation of online visual tracking algorithms.
 You can find the following resources from this site.
  • Benchmark results
  • Dataset with ground-truth annotations
  • Large-scale: 1,550 sequences
  • High-quality: Manual annotation with careful inspection in each frame
Please check out the benchmark details and download links at WATE Benchmark page, evaluation toolkit and sample results at Evaluation and Result

Video

We selected three labeled video sequences from 206 data sets for display. These three videos represent sea, land and air respectively. These datasets are annotated manually, and then the annotated datasets are reviewed. For more data set sequences, please see Benchmark page
