Attributes

We have manually tagged the test sequences with 13 attributes, which represents the challenging aspects in visual tracking.
NAME DESCRIPTION
IV The lighting condition changes during tracking.
OPR The object rotates out of the image plane.
IPR The object rotates in the image plane.
DEF The shape of the target is changing during tracking.
FM The distance between adjacent frames is larger than the target size.
SV The size of the object area is beyond the interval [0.5,2].
CM Camera witching, sudden camera moving
OV The object is disappeared from the target region.
POC The object is partially occluded
FOC The object is fully occluded.
LR The size of the object area is less than 1000 pixels.
SOB Similar object appeared in the background
MB Blurred object region due to camera shaking or fast moving of object.
